Chlone
====

Chlone is a simple tool to clone file permissions.

Get Chlone
--------

Browse the source on GitHub: <http://github.com/yourabi/chlone>

Clone with Git:
    $ git clone git://github.com/yourabi/chlone

Installation
--------
make

make install (or cp chlone /usr/local/bin)
